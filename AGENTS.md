# AGENTS

## Project Goal

Provide a command-line interface that maps 1:1 to the Financial Datasets REST API, exposing endpoints as CLI commands that validate inputs, authenticate with `FINANCIAL_DATASETS_API_KEY`, and print raw JSON responses.

## Typechecking

Run TypeScript typechecking with Bun:

```sh
bun run typecheck
```

## Linting

Run the linter:

```sh
bun run lint
```

Auto-fix lint issues when possible:

```sh
bun run lint:fix
```
